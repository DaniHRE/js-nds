# JS-NDS 🎮

JS-NDS is a small project using the Desmond library to run Nintendo DS (NDS) games directly in the browser.

## How to Use? 👾

1. Choose your game file.
2. Load and play!

It's that simple!

## Play Now 🌐

[Access JS-NDS and start playing!](https://danihre.github.io/js-nds/)
